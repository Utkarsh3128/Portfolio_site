## Steps to use this project locally
* Clone this project
```
git clone https://github.com/Utkarsh2138/Portfolio_site.git
```

* Install Tailwind CSS
```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -y
```

* Install Vite
```
npm install vite
```

* Run Local Server
```
npm run start
```
